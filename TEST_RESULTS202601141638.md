- **Anthropic**: Uses `default_headers` to send Bearer token (SDK defaults to `x-api-key`)


### Authentication Investigation Summary

**Root Cause of Initial Issues:**
- Shell environment had `ANTHROPIC_BASE_URL=https://lamassu.ita.chalmers.se` set
- TypeScript SDK auto-reads environment variables, causing unintended gateway routing
- Isolated test environment revealed true authentication requirements

**Solution:**
- All test files explicitly unset environment variables
- SDKs configured with correct authentication method for gateway
- Verified with curl that gateway accepts `Authorization: Bearer <token>`
