# Open WebUI

[Open WebUI](https://openwebui.portal.chalmers.se){:target="_blank"} is a browser-based chat interface.

## Connect to the Gateway

1. Get your API key from the [Portal](/portal/){:target="_blank"}
2. Follow the [Direct Connections guide](https://docs.openwebui.com/features/chat-conversations/direct-connections/#user-configuration){:target="_blank"}, using:
    - **Base URL**: `https://ai-gateway.portal.chalmers.se/llm/openai/v1`
    - **Auth**: Bearer with your API key
