# OpenAI /v1/models Response Schema

Generated by `./openai-models.sh`

## Top-level structure
```json
{
  "object": "list",
  "data": "array of model objects"
}
```

## Model object fields (all required)
```
id        - string, model identifier
object    - string, always "model"
created   - integer, unix timestamp
owned_by  - string, e.g. "system", "openai"
```
